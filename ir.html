You'll need a Puck.js and an Infrared Receiver Note: If you don't have an IR receiver you may be able to find the 'pronto hex' code for your device online. You can then decode it with this library. use the following to output the contents of the d array rounded to 1 decimal place (which is much more readable): console.log(d.map(a=>a.toFixed(1)).toString()) In fact adding the following code will automatically print the received Infrared code as soon as you have sent it: var lastLen = 0; setInterval(function() { if (d.length && d.length==lastLen) { d.shift(); // remove first element
